# ✅ To-Do Трекер

**Версия:** 1.0.2  
**Язык:** Python 3.10+  
**Хранение данных:** CSV (встроенный модуль `csv`)  
**Тестирование:** pytest + pytest-cov

---

## Описание

Консольное приложение для управления задачами с приоритетами и дедлайнами.
Реализует полный цикл разработки: от ТЗ и TDD до приёмочного тестирования и поддержки.

---

## Структура проекта

```
todo_tracker/
├── data_service.py     # Сервисный слой: CRUD-операции с CSV
├── logic.py            # Бизнес-логика: фильтры, статистика, поиск
├── menu.py             # Консольный интерфейс
├── data/
│   └── tasks.csv       # База данных (создаётся автоматически)
├── tests/
│   ├── test_data_service.py   # Тесты сервисного слоя
│   ├── test_logic.py          # Тесты бизнес-логики
│   └── test_bugfixes.py       # Регрессионные тесты для исправленных багов
├── README.md
├── TEST_CHECKLIST.md
├── SUPPORT_LOG.md
└── REPORT.md
```

## Структура данных (CSV)

| Поле       | Тип  | Описание                             |
|------------|------|--------------------------------------|
| id         | int  | Уникальный идентификатор             |
| title      | str  | Название задачи                      |
| priority   | str  | Приоритет: low / medium / high       |
| deadline   | str  | Дедлайн (YYYY-MM-DD) или пустая строка |
| done       | str  | Статус: True / False                 |
| created_at | str  | Дата создания (YYYY-MM-DD)           |

## Инструкция по запуску

```bash
# 1. Клонировать репозиторий
git clone https://github.com/your-username/todo-tracker.git
cd todo-tracker

# 2. Создать виртуальное окружение (рекомендуется)
python -m venv venv
venv\Scripts\activate   # Windows
source venv/bin/activate  # Linux/Mac

# 3. Установить зависимости
pip install pytest pytest-cov

# 4. Запустить приложение
python menu.py

# 5. Запустить тесты
pytest tests/ -v

# 6. Проверить покрытие кода
pytest tests/ --cov=. --cov-report=term-missing
```

## FAQ

**Q: Как сбросить все задачи?**  
Удалите файл `data/tasks.csv` или выполните:
```python
python -c "import data_service; data_service.clear_all()"
```

**Q: Какие приоритеты поддерживаются?**  
Три уровня: `low` (низкий), `medium` (средний), `high` (высокий).

**Q: В каком формате вводить дедлайн?**  
Строго `YYYY-MM-DD`, например `2025-12-31`.
